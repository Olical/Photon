function index(list,
item)

Returns the index of the item within the string or array.  It will return -1 if not found.

If you only pass a list and no item then it will return true or false depending on whether the list is indexable.

var str = 'photon';
index(str, 'ton'); // 3
index(str, 'foo'); // -1

var arr = [1, 2, 3];
index(arr, 2); // 1
index(arr, 5); // -1

Parameters

listArray or string to search in.
itemItem of sub string to search for.

Returns

The location of the item in the list or -1 if not found.  If you only pass a list then true or false depending on whether it is indexable or not.
